Senior Project

Right now I've set out to create a classic chatbot

The main design structure is 3 sources of information, a processor, and an audio output

Here is the chatbot in action

From start to finish the whole system has around 7-8 seconds of latency for a full response

STT

Speech to Text

For STT I'm using RealtimeSTT
Upon starting the code, the model will always be listening to you and logging what it hears
The log is then handled by the RAG

Current implementation

The RealtimeSTT library makes it a very simple process to get up and running

Currently STT is the only step handled in a seperate file
The file is launched alongside the main file to ensure the STT will always be listening to you

VLM

Vision Language Model

For vision the model I'm using is Pixtral 12b
I am grabbing a snippet of my screen, Downsizing it to 768x432, then feeding it to the model
The model's output is then sent directly to the RAG

Current implementation

Step 1 of using any transformers model is loading into RAM

Here are all the instructions and settings the model uses for inference

A screenshot is taken and ran through the VLM upon calling the function

RAG

Retrieval Augmented Generation

Right now my use of a RAG is certainly not the most complex design but it is all I needed
Right now I am simply pulling chat logs, VLM data, and the STT input for the LLM to respond to

Current implementation

The dataset is checked for new content

Previous chats are pulled to aid in context

All the data is formatted into a single prompt to sent to the LLM

The user prompt and LLM response are logged for future use

LLM

Large Language Model

When it comes to my current system I am using Deepseek chat 7B
It is a model from late 2023 that still performs decently well, and responded well to prompt engineering
I currently have opted to go without a LoRA for sake of completing this project on time, so I am fully relying on my RAG for the formatting of the outputs

Current implementation

The model is also loaded into memory

The prompt is inferenced on by the LLM

Current implementation
